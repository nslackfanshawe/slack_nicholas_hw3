# slack_nicholas_hw3
A to-do list website based on the in-class example and expanded upon.

## Features

- Header with an H1 title styled using an ID selector

- Notebook background image applied to the page

- Three task sections with semantic HTML structure

- Checkbox-based interaction for “In Progress” and “Done” items

- Use of ::after to display icons indicating progress or completion

- Generic selectors, classes, and IDs used throughout the stylesheet

## Technologies Used

HTML5
CSS
Google Fonts
Git and GitHub
